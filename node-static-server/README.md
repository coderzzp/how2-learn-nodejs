# node-static-server
基于node的静态服务器。
## 目录
```
node-static-server
│   README.md
│   app.js
|   static-server.js
|   static-server-pro.js
|   static-server-pro2.js
│
└───public
│   │   fusk.js
│   │   timg.jpeg
│   
└───config
    │   default.json
```
## usage
```cmd
git clone https://github.com/coderzzp/how2-learn-nodejs.git
cd node-static-server
npm i 
pwd
```
粘贴项目所在路径至config文件下defaukt.json,修改root值
此时启动服务
```cmd
node app.js
```
此时cmd会打印端口信息，默认8008,此时你可以在网页上访问目录文件
